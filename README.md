# 🌍 Eco-Audit

## 📝 Descripción
Proyecto realizado para el módulo de **Desarrollo de Aplicaciones Multiplataforma (2º DAM)**. 
Eco-Audit es una herramienta diseñada para auditar el consumo energético o calcular la huella de carbono.

## 🚀 Tecnologías utilizadas
*   **Frontend:** React + Vite
*   **Lenguaje:** JavaScript / JSX
*   **Despliegue:** Vercel

*   https://eco-audit-dun.vercel.app/
